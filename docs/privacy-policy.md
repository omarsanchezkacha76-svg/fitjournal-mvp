# Política de Privacidad

**FitJournal**

Fecha de vigencia: 2026-05-12

## 1. Datos que recopilamos

FitJournal recopila los siguientes datos:
- **Datos de cuenta**: email, nombre de usuario (si creas cuenta)
- **Datos de fitness**: entrenamientos, ejercicios, pesos, repeticiones, comidas, calorías, macros
- **Datos de dispositivo**: información técnica para funcionamiento de la app

## 2. Cómo usamos tus datos

- Para mostrar tu historial de entrenamientos y nutrición
- Para calcular progreso y estadísticas
- Para sincronizar entre dispositivos
- No vendemos tus datos a terceros

## 3. Almacenamiento

Los datos se almacenan en Supabase (EE.UU.) con cifrado. Puedes eliminar tu cuenta y datos en cualquier momento desde la app.

## 4. Tus derechos

- Acceder a tus datos
- Corregir información incorrecta
- Eliminar tu cuenta y datos
- Exportar tus datos

## 5. Contacto

Para ejercer tus derechos: privacy@fitjournal.app

## 6. Cambios

Notificaremos cambios significativos mediante la app o email.
